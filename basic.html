Basic HTML Tags

What are tags

  • HTML markup tags are usually called HTML tags
  • HTML tags are keywords surrounded by angle brackets
  • HTML tags are normally come in pairs,like opening tag b and a closing tag b
  • The first tag in a pair is the starting tag; the second tag is the ending tag
  • The start and end tag are also called opening and closing tags

The basic HTML tags are as follows:
html tag
heading tag- this contains all the heading element(h1,h2,h3,h4,h5,h6)
paragraph tag
body tag-this is where the content of the we appears
head tag-this is where contains title and hyperlinks
hyperlinks-holds the link of web
